# @dpc-sdp/ripple-grants

> Components for displaying Grants information.

--------------------------------------------------------------------------------

## Install

```shell
npm install @dpc-sdp/ripple-grants --save
```

--------------------------------------------------------------------------------

## Dependency graph

```shell
@dpc-sdp/ripple-grants
├── @dpc-sdp/ripple-global
└── moment
```

--------------------------------------------------------------------------------

## Import

```js
import { RplGrantsOverview, RplGrantsListItem } from '@dpc-sdp/ripple-grants'
```
