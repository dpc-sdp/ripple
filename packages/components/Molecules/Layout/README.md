# @dpc-sdp/ripple-layout

> Standard grid-based layouts for structuring pages. Includes skip to content
link.

--------------------------------------------------------------------------------

## Install

```shell
npm install @dpc-sdp/ripple-layout --save
```

--------------------------------------------------------------------------------

## Dependency graph

```shell
@dpc-sdp/ripple-layout
├── @dpc-sdp/ripple-global
├── @dpc-sdp/ripple-grid
└── @dpc-sdp/ripple-icon
```

## Import

```js
import {
  RplBaseLayout,
  RplPageLayout,
  RplQuickExit
} from '@dpc-sdp/ripple-layout'
```
