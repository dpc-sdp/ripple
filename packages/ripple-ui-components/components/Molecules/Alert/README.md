# @dpc-sdp/ripple-alert

> A set of components to display alert notifications.

--------------------------------------------------------------------------------

## Install

```shell
npm install @dpc-sdp/ripple-alert --save
```

--------------------------------------------------------------------------------

## Dependency graph

```shell
@dpc-sdp/ripple-alert
├── @dpc-sdp/ripple-global
├── @dpc-sdp/ripple-icon
└── @dpc-sdp/ripple-link
```

--------------------------------------------------------------------------------

## Import

```js
import { RplAlert, RplAlertBase } from '@dpc-sdp/ripple-alert'
```
