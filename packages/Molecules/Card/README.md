# @dpc-sdp/ripple-card

> Card-themed containers for content such as events and dates, navigation,
featured, promotional. Recommended implementation using the grid component.

--------------------------------------------------------------------------------

## Install

```shell
npm install @dpc-sdp/ripple-card --save
```

--------------------------------------------------------------------------------

## Dependency graph

```shell
@dpc-sdp/ripple-card
├── @dpc-sdp/ripple-global
├── @dpc-sdp/ripple-icon
└── @dpc-sdp/ripple-link
```
