# @dpc-sdp/ripple-sitemap

> A component to display nested, navigational links.

--------------------------------------------------------------------------------

## Install

```shell
npm install @dpc-sdp/ripple-sitemap --save
```

--------------------------------------------------------------------------------

## Dependency graph

```shell
@dpc-sdp/ripple-sitemap
├── @dpc-sdp/ripple-global
└── @dpc-sdp/ripple-link
```

--------------------------------------------------------------------------------

## Import

```js
import RplSitemap from '@dpc-sdp/ripple-sitemap'
```
