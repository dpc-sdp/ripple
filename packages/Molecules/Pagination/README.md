# @dpc-sdp/ripple-pagination

> A component for displaying a pagination bar with controls to change current
step.

--------------------------------------------------------------------------------

## Install

```shell
npm install @dpc-sdp/ripple-pagination --save
```

--------------------------------------------------------------------------------

## Dependency graph

```shell
@dpc-sdp/ripple-pagination
├── @dpc-sdp/ripple-global
└── @dpc-sdp/ripple-icon
```
