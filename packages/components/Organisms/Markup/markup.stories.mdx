# Markup

## Installation

```bash
npm install @dpc-sdp/ripple-markup --save
```

## Import

```javascript
import { RplMarkup } from '@dpc-sdp/ripple-markup'
```

## Usage
````html
<rpl-markup
  :html="html"
  :plugins="[]"
  :options="{}"
/>
````
