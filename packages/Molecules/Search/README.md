# @dpc-sdp/ripple-search

> Components for generating a search field with filters, and search results.

--------------------------------------------------------------------------------

## Install

```shell
npm install @dpc-sdp/ripple-search --save
```

--------------------------------------------------------------------------------

## Dependency graph

```shell
@dpc-sdp/ripple-search
├── @dpc-sdp/ripple-form
├── @dpc-sdp/ripple-global
├── @dpc-sdp/ripple-icon
└── @dpc-sdp/ripple-meta-tag
```
