@dpc-sdp/ripple-share-this
============

> A component for displaying social media buttons to share articles. Recommended for use in sidebars.


--------------------------------------------------------------------------------


## Install


```shell
npm install @dpc-sdp/ripple-share-this --save
```


--------------------------------------------------------------------------------


## Dependency graph

```shell
@dpc-sdp/ripple-share-this
├── @dpc-sdp/ripple-global
└── @dpc-sdp/ripple-icon
```
