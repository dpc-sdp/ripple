# @dpc-sdp/ripple-related-links

> A component for displaying internal / external links related to an article.
Recommended for use in sidebars.

--------------------------------------------------------------------------------

## Install

```shell
npm install @dpc-sdp/ripple-related-links --save
```

--------------------------------------------------------------------------------

## Dependency graph

```shell
@dpc-sdp/ripple-related-links
├── @dpc-sdp/ripple-global
└── @dpc-sdp/ripple-link
```
