# @dpc-sdp/ripple-timeline

> A timeline component for displaying sequential items.

--------------------------------------------------------------------------------

## Install

```shell
npm install @dpc-sdp/ripple-timeline --save
```

--------------------------------------------------------------------------------

## Dependency graph

```shell
@dpc-sdp/ripple-timeline
├── @dpc-sdp/ripple-global
└── @dpc-sdp/ripple-link
```

--------------------------------------------------------------------------------

## Import

```js
import { RplTimeline } from '@dpc-sdp/ripple-timeline'
```
