# @dpc-sdp/ripple-map

> An interactive map component

--------------------------------------------------------------------------------

## Install

```shell
npm install @dpc-sdp/ripple-map --save
```

--------------------------------------------------------------------------------

## Dependency graph

```shell
@dpc-sdp/ripple-map
├── @dpc-sdp/ripple-global
├── ol
└── ol-ext
```

--------------------------------------------------------------------------------

## Import

```js
import { RplMap } from '@dpc-sdp/ripple-map'
```
