# @dpc-sdp/ripple-icon

> A configurable SVG, sprite-based icon set for use with ripple components.

--------------------------------------------------------------------------------

## Install

```shell
npm install @dpc-sdp/ripple-icon --save
```

--------------------------------------------------------------------------------

## Dependency graph

```shell
@dpc-sdp/ripple-icon
└── @dpc-sdp/ripple-global
```
