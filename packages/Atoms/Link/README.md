# @dpc-sdp/ripple-link

> Components for creating page links in plain text, and with accompanying icon.
External links open in a new window by default.

--------------------------------------------------------------------------------

## Install

```shell
npm install @dpc-sdp/ripple-link --save
```

--------------------------------------------------------------------------------

## Dependency graph

```shell
@dpc-sdp/ripple-link
├── @dpc-sdp/ripple-global
└── @dpc-sdp/ripple-icon
```
