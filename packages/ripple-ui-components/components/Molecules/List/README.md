# @dpc-sdp/ripple-list

> List component for displaying an icon and text in a list.

--------------------------------------------------------------------------------

## Install

```shell
npm install @dpc-sdp/ripple-list --save
```

--------------------------------------------------------------------------------

## Dependency graph

```shell
@dpc-sdp/ripple-list
├── @dpc-sdp/ripple-global
├── @dpc-sdp/ripple-icon
└── @dpc-sdp/ripple-link
```

--------------------------------------------------------------------------------

## Import

```js
import RplList from '@dpc-sdp/ripple-list'
```
