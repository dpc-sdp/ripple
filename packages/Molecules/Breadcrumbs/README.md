# @dpc-sdp/ripple-breadcrumbs

> A compoennt to display the active path, with links. Does not display on small
screens.

--------------------------------------------------------------------------------

## Install

```shell
npm install @dpc-sdp/ripple-breadcrumbs --save
```

--------------------------------------------------------------------------------

## Dependency graph

```shell
@dpc-sdp/ripple-breadcrumbs
├── @dpc-sdp/ripple-global
└── @dpc-sdp/ripple-link
```
