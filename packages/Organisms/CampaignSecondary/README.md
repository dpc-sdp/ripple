# @dpc-sdp/ripple-campaign-secondary

> A component to showcase secondary content previews. Should be used along with
Campaign Primary.

--------------------------------------------------------------------------------

## Install

```shell
npm install @dpc-sdp/ripple-campaign-secondary --save
```

--------------------------------------------------------------------------------

## Dependency graph

```shell
└── @dpc-sdp/ripple-global
```
