# @dpc-sdp/ripple-button

> A button themed element. Will be a `<button>` by default, or a an `<a>` if a
`href` is provided.

--------------------------------------------------------------------------------

## Install

```shell
npm install @dpc-sdp/ripple-button --save
```

--------------------------------------------------------------------------------

## Dependency graph

```shell
@dpc-sdp/ripple-button
├── @dpc-sdp/ripple-global
└── @dpc-sdp/ripple-link
```

--------------------------------------------------------------------------------

## Import

```js
import RplButton from '@dpc-sdp/ripple-button'
```
