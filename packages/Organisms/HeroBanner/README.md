# @dpc-sdp/ripple-hero-banner

> A component to display a page heading, subtext and useful links.

--------------------------------------------------------------------------------

## Install

```shell
npm install @dpc-sdp/ripple-hero-banner --save
```

```shell
└── @dpc-sdp/ripple-link
```
