# @dpc-sdp/ripple-site-header

> Site header for providing site-wide navigation and search on all pages.

--------------------------------------------------------------------------------

## Install

```shell
npm install @dpc-sdp/ripple-site-header --save
```

--------------------------------------------------------------------------------
