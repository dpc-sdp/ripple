# @dpc-sdp/ripple-grid

> A grid system for page layout.

--------------------------------------------------------------------------------

## Install

```shell
npm install @dpc-sdp/ripple-grid --save
```

--------------------------------------------------------------------------------

## Dependency graph

```shell
@dpc-sdp/ripple-grid
└── @dpc-sdp/ripple-global
```

--------------------------------------------------------------------------------

## Import

```js
import { RplContainer, RplRow, RplCol } from '@dpc-sdp/ripple-grid'
```
